meowdopt API

  • It is a cat API seeded with a list of cats and shelters from Alberta, Canada.

Routes

Shelters

  • GET /shelters
  • GET /shelters/:id
  • POST /shelters/:id/cats
  • DELETE /cats/:id/remove-like

Cats

  • GET /cats
  • GET /cats/:id
  • POST /cats/:id/like
  • DELETE /cats/:id/remove-like

Users

  • GET /users
  • GET /users/:id
  • GET /users/:id/favorites
  • GET /users/:id/requests
  • POST /users/signup

Login

  • POST /login
